# 🔐 CipherNest

> A lightweight, modular cryptography toolkit written in Python.

CipherNest provides secure encryption, hashing, digital signatures, and key management utilities with a clean CLI and extensible architecture.

It is designed for developers, security enthusiasts, and students who want a practical and understandable cryptographic toolbox.

---

## 🚀 Features

- AES-256 encryption (AES-GCM)
- RSA-4096 asymmetric encryption
- RSA digital signatures
- Argon2 secure password hashing
- CLI interface
- Modular architecture
- Secure key generation
- Extensible design
